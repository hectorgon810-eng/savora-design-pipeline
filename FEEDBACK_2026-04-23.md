## GLOBAL RULES (apply to ALL brands)

1. **Identity lock — no image alteration.** Reference photos pass through unchanged. No AI reinterpretation of the dish itself.
2. **No logo redesign.** If brand logo appears drawn, it must be the EXISTING logo rendered — not a reinvention.
3. **No fake menus / holiday specials / seasonal claims.** Stick to regular menu designs. No "upcoming X", no holiday bundles unless verified with Hector first.
4. **No arrows on obvious items.** Meat / steak / rice / beans / lettuce / cheese — obvious. Only label with arrows when the dish has small non-obvious details (sauces, garnishes worth naming). Machine mislabels small items; safer to skip the callout.
5. **Typography target: match the actual logo weight.** Not chunkier. Letters should feel like the brand's own logo — not a generic bold sans. 4★ Azteca tier nailed this; 2★ failed.
