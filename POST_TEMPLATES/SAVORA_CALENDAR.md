## How to use this file

- **Brand column** tells you which of the three restaurants should post — `JH` Jackson House, `BM` Blue Mezcal, `AZ` El Azteca (both locations). "All" = all three.
- **Template** maps to the post playbook (A1 Product, B1 Recurring, C1 Seasonal, D1 Event, E1 Operational, F1 Evergreen).
- **Lead** is how many days BEFORE the date you should post the "coming up" announcement. Day-of post is separate.
- **Priority:** ⭐⭐⭐ = must post · ⭐⭐ = post if bandwidth · ⭐ = optional / promote only if relevant

| Event size | Teaser post | Day-of post | Follow-up |
|---|---|---|---|
| Major holiday (Cinco, Valentine's, Mother's Day) | 3–5 days before | Day-of | — |
| Secondary holiday (Memorial, Labor, July 4) | 2–3 days before | Day-of | — |
| Minor (Margarita Day, Tequila Day) | 1–2 days before | Day-of | — |
| Brand event (menu launch, live music) | 7–10 days before | Day-of | Next-day recap |
| Heritage month (Hispanic Heritage Sep 15–Oct 15) | Opening-day post | Weekly feature | Closing-day post |

## Content rotation per holiday (so you don't post the same thing twice)

For every major holiday, prep **3 content slots**:
1. **Teaser** (3–5 days out) — "coming up" graphic, dated, benefit-led
2. **Day-of** (morning or early afternoon) — festive graphic + one food/drink hero
3. **Recap** (optional, day after) — UGC or BTS photo from the day

Each slot uses a different template:
- Teaser → C1 Seasonal (graphic-only)
- Day-of → A1 Product with C1 Seasonal accent (mixed)
- Recap → A1 Product photo + script overlay
