## Palette — inverted from the earlier take

Primary direction: **black / deep-charcoal ground + brass-foil gradient type + deep tonal forest/oxblood fields**. Cream appears only as accent body-type colour. The old "60 cream" rule is wrong — corrected 2026-04-21 after Hector feed analysis.

| Role | Hex | Name | Usage |
|---|---|---|---|
| Canvas | `#0A0A0A` → `#1A1612` | Charred black / deep charcoal | Default ground, always with a 5–10% tonal gradient (never flat) |
| Hero type | `#C9A24B` → `#B08937` → `#C9A24B` | Brass gradient foil | Headlines — 3-stop gradient with specular hotspot, reads as foil catching light |
| Field A | `#1F3B2D` → `#0F1B15` | Forest → near-black gradient | Deep tonal field (darker at the bottom), never a flat green |
| Field B | `#5B1A1A` → `#2A0B0B` | Oxblood → deep wine gradient | Same treatment — deep, never flat |
| Body type | `#F2EADF` | Warm cream | Accent body type only, never the canvas |
| Rare accent | `#C8962F` | Mustard | Occasional sub-accent — less than 5% of layout |
| Walnut | `#3B2A1E` | Walnut brown | Frame / rule / accent |

**Rule:** 60 black/charcoal ground, 30 brass-foil type + one deep tonal field (forest OR oxblood), 10 cream body type. Mustard optional accent <5%.

**Reference:** the National Cocktail Day Jackson House flyer (gold foil on black) is the exact target aesthetic. Foil-stamped elegance, not rustic.

## Typography

| Role | Fonts | Prompt descriptor |
|---|---|---|
| Display | Farnham · Sentinel · Caslon · Cheltenham · Caponi | "old-style or slab serif with wood-type character" |
| Sans | Knockout · Trade Gothic Condensed · Marr Sans | "condensed all-caps grotesque labels with hairline rules" |
| Script | chalk-lettering for specials only | "chalkboard script (specials only, sparingly)" |

Sizes: display 120–200px · subhead 48–72px · body 28–36px.
Tracking: display tight (−10 to −30 em), all-caps wide (+40 to +120 em).
Leading: display 1.05–1.15 · body 1.3–1.45.

## Visual language

- Cream laid paper texture
- Letterpress emboss on type (letter sits as if pressed into the paper)
- Film grain (Portra 800) on all photography
- Warm tungsten candlelight 2800K
- Shallow depth of field, human hands in frame
- Documentary 35mm feel for BTS
- Hairline rules separating sections
- Vintage menu reproductions, crests, monograms for event/award moments
- Scratchboard / engraving illustration (carrot, herb, spoon) as occasional graphic device

## Voice & layout

**Structure:** `[Dish Name]. [emoji]\n\n[One punchy line.]\n\n📍 17 Wood St, Middletown, DE 19709\nReserve your table | link in bio 📍`

**Rules:** Classical, warm, telegraphic, confident. One punchy line — let the image do the work. Never explain the dish unless it's new.

**Voice examples:**
- `Dilly Dally Dirty Martini. 🍸 / You already know.`
- `Burrata. 🍅 / Some things don't need to be complicated.`
- `Cornbread Muffins. 🧈 / Warm out of the oven. Butter melts on contact.`
